# CodeNote (코드노트)

CodeNote는 안드로이드 애플리케이션을 기획·개발·배포하는 개인사업자 개발 브랜드입니다.  
실사용 환경을 기준으로 한 유틸리티 및 관리 목적의 모바일 앱을 개발하고 있습니다.

---

## 개발 서비스

- **포켓 블랙박스**
  - 개인 안전 및 기록 목적의 안드로이드 애플리케이션

- **공신폰 만들기**
  - 스마트폰 사용 제한 및 관리 기능을 제공하는 학습·관리용 앱

- **홍보 어플 제작**
  - 매장 및 서비스 홍보 목적의 모바일 애플리케이션 제작

- **기타 개발 영역**
  - 안드로이드 앱 개발
  - 관리 및 유틸리티 앱 제작
  - 앱 기획 및 상용화
