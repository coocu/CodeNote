개인정보 처리방침

Mobile Screen SYNC(이하 "앱")은 사용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 및 관련 법령을 준수합니다.
본 개인정보 처리방침은 앱에서 어떤 정보가 수집되고 어떻게 사용되는지를 설명합니다.

1. 수집하는 개인정보

앱은 회원가입 기능이 없으며, 어떠한 개인정보도 서버로 전송하거나 저장하지 않습니다.

2. 사용되는 권한

앱은 다음 기능을 위해 일부 기기 권한을 사용할 수 있습니다.

화면 캡처 권한 (MediaProjection)

기능: 다른 기기로 화면 미러링 기능 제공

설명: 사용자가 명시적으로 동의한 경우에만 화면 캡처가 실행됩니다.

Wi-Fi 및 네트워크 권한

기능: Wi-Fi Direct를 통한 기기간 연결

설명: 동일 네트워크 또는 Wi-Fi Direct 연결을 위해 사용됩니다.

카메라 권한

기능: USB HDMI 캡처카드 영상 표시

설명: USB로 연결된 외부 캡처 장치의 영상을 표시하기 위해 사용됩니다.

3. 데이터 처리 방식

앱에서 전송되는 화면 데이터는 다음과 같이 처리됩니다.

데이터는 사용자의 기기 간 직접 연결(P2P)을 통해 전송됩니다.

데이터는 외부 서버로 업로드되거나 저장되지 않습니다.

모든 데이터 처리는 사용자의 기기 내에서만 이루어집니다.

4. 제3자 제공

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

5. 데이터 저장

앱은 개인정보를 저장하지 않으며,
사용자의 화면 데이터 또한 앱 외부에 저장되지 않습니다.

6. 개인정보 보호

본 앱은 사용자 데이터를 서버에 저장하지 않는 구조로 설계되어 있으며,
모든 기능은 사용자 기기 간 직접 연결 방식으로 동작합니다.

7. 정책 변경

본 개인정보 처리방침은 필요에 따라 변경될 수 있으며,
변경 시 앱 업데이트 또는 공지를 통해 안내됩니다.
