📄 개인정보 처리방침



스미싱 그만! (Smishing Blocker)

최종 업데이트: 2025-01-00



본 애플리케이션(이하 "앱")은 사용자 개인정보를 매우 중요하게 생각하며, 「개인정보보호법」 및 관련 법령을 준수합니다.

앱은 스미싱 문자 분석, 알림 텍스트 분석 기능을 제공하기 위해 아래와 같은 방식으로 최소한의 정보만을 처리합니다.



1\. 수집하는 개인정보 항목



본 앱은 아래의 민감하지 않은 데이터를 기기 내부에서만 분석하며, 외부 서버로 전송하지 않습니다.



📌 (1) 문자 메시지 내용 (READ\_SMS)



사용 목적: 스미싱 위험도 분석



보관 여부: 앱 내부 로컬 분석 즉시 처리, 저장하지 않음



외부 전송: 없음



📌 (2) 알림 내용 (POST\_NOTIFICATIONS)



사용 목적: 사용자가 붙여 넣은 알림 텍스트 분석



보관 여부: 즉시 분석 후 화면에 표시만 함



외부 전송: 없음



2\. 개인정보 수집 방법



사용자가 앱 기능을 사용할 때 문자 메시지 읽기 권한을 허용한 경우



사용자가 직접 화면에 텍스트를 입력/붙여넣은 경우



앱은 사용자의 동의 없이 어떠한 개인정보도 수집하거나 전송하지 않습니다.



3\. 개인정보의 보유 및 이용 기간



본 앱은 개인정보를 아예 저장하지 않으며,

분석 과정은 \*\*오프라인(기기 내부)\*\*에서 실시간 수행됩니다.



4\. 개인정보 제3자 제공



앱은 개인정보를 어떤 형태로든 외부로 제공하거나 공유하지 않습니다.



서버 전송 없음



원격 저장 없음



광고 SDK 없음



제휴 공유 없음



5\. 개인정보 처리 위탁



앱은 어떤 회사/개인에게도 개인정보 처리를 위탁하지 않습니다.



6\. 사용자의 권리



앱이 개인정보를 저장하지 않기 때문에 사용자의 데이터는 모두

기기 내 기본 문자 메시지 앱을 통해 사용자 스스로 관리할 수 있습니다.



7\. 개인정보의 파기



앱은 개인정보를 수집 및 저장하지 않으므로

파기해야 할 데이터가 존재하지 않습니다.



8\. 권한 사용 안내



앱은 아래의 권한만 사용하며, 다른 권한은 요구하지 않습니다.



📌 READ\_SMS



문자 메시지를 읽어 스미싱 위험도를 분석함



분석 목적 외 사용 없음



서버 전송 없음



📌 POST\_NOTIFICATIONS



사용자가 알림 내용을 복사하여 앱에 넣을 때 필요한 권한



외부 전송 없음
